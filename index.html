Om oss

Denna hemsida skapades för att visa upp möjligheten att spela video med enkla kontroller och moderna funktioner. Vårt mål var att skapa en lättanvänd videospelare som fungerar direkt i webbläsaren.

Sidan är byggd med HTML, CSS och JavaScript – utan att använda några tunga ramverk eller servrar. Allt körs direkt i din webbläsare.
